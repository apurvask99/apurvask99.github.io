Fabflix

Project information

  • Category: Web Dev
  • Built For: Project in Web Development and Databases Course
  • Project dates: March 2020 - June 2020

I created a full-stack web application where users would be able to browse or search movies and stars based on various parameters such as movie title, stars, year, director and genres. Users also had the option to "buy" movies. The frontend was developed using HTML5, CSS, and JavaScript while the backend was developed with Java/JDBC and mySQL for database. The application was hosted on an Apache Tomcat server on top of an AWS EC2 instance as well as Google Cloud Platform instance. An Android application for the website was also created using Android Studio.
