Diabetes Modeling

Project information

  • Category: Machine Learning
  • Built For: Machine Learning and Data Mining Course
  • Project date: March 2020
  • Project: Link

In this project, we were given a large CSV file dataset of diabetes patients and we had to determine using machine learning techniques whether they would be readmitted in less than 30 days, greater than 30 days, or not readmitted. We utilized machine learning libraries scikit-learn and mltools to process the data and used the Random Forest model to predict whether the patient would be readmitted in less than 30 days, greater than 30 days, or not readmitted.
