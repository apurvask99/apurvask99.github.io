I helped create this web application as a full-stack developer for a STEM education program for school children called Dreams For Schools. This application allows for administrators to assign instructors to students for their various STEM programs such as AppJam+ (building apps) based on parameters such as region and schedule. The backend was written in Python which used Firebase Realtime database and the frontend made use of the React.js framework to display the data.
